# Profile Screen

A simple user profile screen built with Flutter. This app displays a user's name, photo, and bio.

## Features

- Display user's profile picture
- Show user's name and job title
- Display a brief bio

## Getting Started

### Prerequisites

- Flutter SDK installed on your machine
